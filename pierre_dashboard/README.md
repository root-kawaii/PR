# Pierre Dashboard

Vite + React dashboard for club owners to manage venues, events, tables, reservations, and check-ins.

## Setup

```bash
npm install
npm run dev
```

## Optional analytics env vars

- `VITE_POSTHOG_KEY`
- `VITE_POSTHOG_HOST` (defaults to `https://eu.i.posthog.com`)
- `VITE_APP_ENV`
